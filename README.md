###　docker上での起動
docker上でnginx,app,mysqlコンテナを構築しappを起動する
```
docker-compose up
```

###　local開発環境での起動
docker上でmysqlコンテナを構築
```
docker-compose -f docker-compose-local.yml up -d
```
app起動
```
./gradlew bootRun
```
test
```
./gradlew test
```
lint
```
# Check
./granlew ktlintCheck

# Format(Caution!)
._grandlew ktlintFormat
```
And you can run app from intelliJ IDEA
